Internal Company Dashboard - UI Design & Development

Dev-Ops Dashboard

I co-led the design and development of a dashboard for our DevOps team to monitor deployment details and project costs for Barclays Mobile Banking's backend. My primary focus was on creating an intuitive but visually appealing user interface, ensuring a seamless user experience. Additionally, I collaborated closely with the development of the product to integrate essential functionalities. This project significantly improved our team's efficiency and provided valuable insights for Barclays development operations.

Project Goals

Outlining the primary objectives for the dashboard.

Display a Comprehensive Data Overview
Provide Real-Time Data & Information
Redesign User Interface into a Clean/Modern Aesthetic.

II

User Research

User research gathered to gain insight into the pain points developers face with the legacy dashboards.

Behaviours

  • Monitoring Deployment Status
  • Version Control and Rollout
  • Accessing Logs and Metrics
  • Performance Testing and Validation
  • Health Checks and Post-deployment Validation

Pain Points

  • Complexity and Usability
  • Lack of Real-time Updates
  • Inadequate Error Information
  • Integration Issues
  • Accommodation Conflicts
  • Performance Overheads

Desires

  • User-friendly Interface
  • Real-time Monitoring
  • Comprehensive Error Reporting
  • Seamless Integration
  • Customisable Dashboards
  • Historical Data and Trends
  • Automated Alerts and Notifications
  • Visualisation Tools
  • Documentation and Help Resources
